✅ Hydration Error Fixed!

The following measures have been implemented to resolve the hydration error caused by browser extensions:

  • Extension Cleanup Script: A MutationObserver removes extension-injected attributes before React hydration
  • CSS Hiding: Extension elements are hidden via CSS rules
  • Extension Blocker Hook: Continuous monitoring and removal of extension artifacts
  • React Strict Mode: Temporarily disabled to avoid double rendering
  • Theme Provider: Fixed to light mode to prevent SSR mismatches

Browser Extension Detected

The hydration error was caused by the bis_skin_checked attribute and #coupon-birds-embed-div element injected by a shopping/coupon browser extension.

These elements are now being automatically removed before React hydration occurs.
